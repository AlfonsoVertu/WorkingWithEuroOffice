# Virtual World Gateway HA: Double Branded Edition

A premium suite of Home Assistant add-ons for networking, office automation, and AI integration.

## 🚀 Add-ons Included
- **Virtual World Dashboard**: WordPress REST Middleware for AI (WP GPT Automation Pro).
- **EuroOffice**: Collaborative Document Server (ONLYOFFICE).
- **Nextcloud**: File storage with integrated ONLYOFFICE & OCR.
- **Odoo 16**: ERP and Business Applications.
- **Nginx Proxy Manager**: Easy SSL & Proxy management.
- **AdGuard Home**: Network-wide ad & tracker blocking.
- **Elasticsearch**: Powerful search and analytics engine.
